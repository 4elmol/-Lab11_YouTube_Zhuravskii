## Описание
Проект представляет собой адаптивную верстку главной страницы YouTube. В ходе работы использовал HTML, CSS, Flexbox и Grid для создания сетки видео. Реализована адаптивная навигация, чипсы с категориями и карточки видео, которые подстраиваются под разные разрешения экрана.
---
## Реализованные функции
- [ ] Адаптивный хедер с поиском
- [ ] Боковая панель навигации
- [ ] Категории (чипсы) с интерактивностью
- [ ] Сетка видео с карточками
- [ ] Hover-эффекты на карточках
